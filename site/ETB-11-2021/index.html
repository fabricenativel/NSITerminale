    Question 1. (a) : 2 pts

    def total_hors_reduction(tab):
    '''Calcul la somme des éléments de tab'''
    assert type(tab)==list, "L'argument doit être une liste"
    thr = 0
    for prix in tab:
        assert type(prix)==int or type(prix)==float,"Les prix doivent être de types numériques"
        thr += prix
    return thr

    Question 1. (b) : 1 pt

    def offre_bienvenue(tab):
    """ tableau -> float """
    somme =0
    longueur = len(tab)
    if longueur > 0:
        somme = tab[0]* 0,8
    if longueur > 1:
        somme = somme + tab[1] * 0,7
    if longueur > 2:
        for i in range(2,longueur):
            somme = somme + tab[i]
    return somme

    Question 2.a : 1 pt

    Deux cases sont adjacentes si leur lignes ou colonnes diffèrent d'une unité. C'est un ou exclusif, c'est à dire soit les lignes diffèrent d'une unité, soient les colonnes mais pas les deux. Comme la différence est \(+1\) ou \(-1\) selon le sens de la soustraction l'élévation au carré permet de se ramener à un et donc de comparer d==1
